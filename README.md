### 제목

일반적으로 #을 쓰면 글씨가 커짐. #이 여러 개 쓰일 때마다 글씨가 작아짐. 1개는 너무 크기 때문에 2~3개짜리를 제목으로 많이 선택함.



소스코드 블록은 숫자 1 왼쪽에 있는 특수 문자를 사용해서 작성 ```언어 ```

```c
#include <stdio.h>

int main(void) {
  printf("Hello world");
  return 0;
}
```
